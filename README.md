# Pandas Learning Repository

## Overview

This repository contains Python programs that demonstrate fundamental and advanced concepts of the Pandas library. It is designed as a learning resource for beginners who want to build strong data analysis and data manipulation skills using Python.

The repository is divided into two sections:

* **Basic** – Introduction to Pandas and dataset exploration.
* **Advance** – Data cleaning, transformation, and manipulation techniques.

## Basic Topics

The Basic folder covers the core concepts of Pandas:

### Files

* **intro.py** – Introduction to Pandas Series and DataFrames.
* **reading_data.py** – Reading data from CSV, Excel, and JSON files.
* **exploring_dataset.py** – Exploring datasets using common Pandas functions.
* **information_about_dataset.py** – Viewing dataset information and statistics.
* **shape_colmn.py** – Understanding dataset dimensions and column operations.

### Concepts Covered

* Pandas Basics
* Series and DataFrames
* Reading Different File Formats
* Dataset Exploration
* Data Inspection
* Rows and Columns Operations

---

## Advance Topics

The Advance folder focuses on real-world data preprocessing and manipulation techniques.

### Files

* **concat.py** – Combining multiple DataFrames.
* **delete_col.py** – Removing columns from a DataFrame.
* **group_by.py** – Grouping data and performing aggregations.
* **handling_missing_values.py** – Detecting and handling missing values.
* **inserting_new_col.py** – Adding new columns to a DataFrame.
* **interpolation.py** – Filling missing values using interpolation methods.
* **sorting.py** – Sorting rows and columns.
* **updating_value.py** – Updating and modifying DataFrame values.

### Concepts Covered

* Data Cleaning
* Missing Value Handling
* Data Transformation
* Data Aggregation
* GroupBy Operations
* DataFrame Concatenation
* Sorting and Filtering
* Column Manipulation

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/pandas-learning.git
```

Navigate to the project directory:

```bash
cd pandas-learning
```

Install dependencies:

```bash
pip install pandas numpy
```
